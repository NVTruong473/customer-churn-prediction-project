
# Customer Churn Prediction Project

A Machine Learning project that predicts whether a customer is likely to leave a service based on customer behavior, billing details, and subscription history.

Dự án Machine Learning dự đoán liệu khách hàng có khả năng rời bỏ dịch vụ dựa trên hành vi sử dụng, thông tin thanh toán và lịch sử đăng ký.

---

## Project Overview

Customer churn is one of the most important business problems for subscription-based companies.

Churn khách hàng là bài toán quan trọng đối với doanh nghiệp thuê bao.

This project builds a complete end-to-end Machine Learning pipeline.

Dự án xây dựng quy trình Machine Learning hoàn chỉnh.

---

## Final Model

Tuned Random Forest

Best ROC-AUC: 0.8437

---

## Models Compared

- Logistic Regression
- Decision Tree
- Random Forest
- KNN
- SVM

---

## Key Insights

- Long-term customers are less likely to churn
- Short-term contracts have higher churn risk
- Higher monthly charges may increase churn probability
- Support services improve retention
